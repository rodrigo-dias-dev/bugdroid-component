# 🚀 Overview

A proposta deste projeto foi recriar uma página temática sobre o mascote do Android utilizando uma abordagem moderna com HTML5 e CSS3.

A V2 representa uma evolução importante da arquitetura do projeto, trazendo melhorias na organização semântica, estrutura visual e refinamento responsivo.

Durante o desenvolvimento, o foco principal foi compreender melhor:

* Flexbox;
* Box Model;
* largura de leitura;
* hierarquia visual;
* composição de containers;
* organização semântica;
* refinamento de spacing;
* responsividade progressiva;
* componentização estrutural.

## 🧠 What I Learned

Este projeto marcou uma grande evolução no meu processo de aprendizado em Front-End.

Durante o desenvolvimento da V2 comecei a entender de forma muito mais clara como os elementos se relacionam estruturalmente dentro do layout. O entendimento do Box Model foi uma das principais viradas de chave até aqui.

Depois disso, muitos conceitos começaram a fazer sentido de forma mais natural.

Alguns conceitos que aprofundei durante esta versão:

* Box Model;
* Flexbox;
* CSS Grid;
* composição de layout;
* largura de leitura;
* Mobile First;
* Media Queries;
* hierarquia visual;
* arquitetura CSS;
* separação entre estrutura e conteúdo;
* organização semântica;
* refinamento tipográfico;
* distribuição de espaçamento;
* componentização visual.

Outro ponto extremamente importante foi o uso constante do DevTools durante o processo. Aprender a inspecionar elementos, testar ajustes em tempo real e entender o comportamento visual do layout mudou completamente minha forma de estudar e debugar interfaces.

### Exemplo de composição textual

```css
.header-text {
    width: min(22rem, 100%);
    margin: 0 auto;
}
```

Essa abordagem ajudou a criar uma largura de leitura mais confortável e uma composição textual mais equilibrada em dispositivos móveis.
